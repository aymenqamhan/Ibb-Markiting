هذا الجدول يخزن بيانات أوامر الشراء بشكل عام، كل سجل فيه يمثل طلب شراء من مورد معين.

| الحقل                    | الوصف                                                              |
| ------------------------ | ------------------------------------------------------------------ |
| `id`                     | المعرف الفريد لأمر الشراء (رقم تسلسلي تلقائي).                     |
| `supplier_id`            | معرف المورد الذي تم إصدار أمر الشراء له (مرتبط بجدول `suppliers`). |
| `order_date`             | تاريخ ووقت إصدار أمر الشراء.                                       |
| `expected_delivery_date` | التاريخ المتوقع لوصول المنتجات المطلوبة.                           |
| `actual_delivery_date`   | التاريخ الفعلي لوصول المنتجات (يُحدث عند الاستلام).                |
| `total_amount`           | المبلغ الإجمالي لأمر الشراء (مجموع تكلفة كل الأصناف).              |
| `status`                 | حالة أمر الشراء، مثل مسودة، قيد الطلب، مكتمل، ملغي، إلخ.           |
| `notes`                  | ملاحظات إضافية متعلقة بأمر الشراء.                                 |
| `created_by_user_id`     | معرف المستخدم الذي أنشأ أمر الشراء (مرتبط بجدول المستخدمين).       |
| `created_at`             | تاريخ ووقت إنشاء السجل تلقائيًا.                                   |
| `updated_at`             | تاريخ ووقت آخر تحديث للسجل تلقائيًا.                               |
